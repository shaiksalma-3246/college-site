Experiment 1: Introduction to SPSS

Aim:

To understand basic SPSS operations such as sorting data, splitting files, computing new variables, recoding values, and selecting cases.
Description:

In this experiment, fundamental data handling operations commonly used in SPSS are implemented using Python in Google Colab with the help of the pandas library. A sample dataset containing student information such as ID, age, gender, and score is created and analyzed. The dataset is first organized using the Sort File operation to arrange records based on scores. The Split File operation is applied to separate the data according to gender for independent analysis. A new variable is generated using the Compute operation by modifying existing score values. The Recode operation is used to convert numerical scores into categorical outcomes such as Pass or Fail. Finally, the Select Cases operation is performed to filter records that satisfy a given condition. These operations are essential for data preparation and analysis in predictive analytics.
Code :

import pandas as pd
data = {
    "ID": [1, 2, 3, 4, 5, 6],
    "Age": [18, 20, 19, 21, 22, 20],
    "Gender": ["Male", "Female", "Male", "Female", "Male", "Female"],
    "Score": [45, 78, 60, 90, 35, 55]
}

df = pd.DataFrame(data)
print("Original Data:")
print(df)
sorted_df = df.sort_values(by="Score")
print("\nSorted Data by Score:")
print(sorted_df)
print("\nSplit Data by Gender:")
male_df = df[df["Gender"] == "Male"]
female_df = df[df["Gender"] == "Female"]
print("\nMale Students:")
print(male_df)
print("\nFemale Students:")
print(female_df)
df["Final_Score"] = df["Score"] + 5
print("\nAfter Computing Final_Score:")
print(df)
df["Result"] = df["Score"].apply(lambda x: "Pass" if x >= 50 else "Fail")
print("\nAfter Recoding Pass/Fail:")
print(df)
selected_df = df[df["Score"] >= 60]
print("\nSelected Cases (Score >= 60):")
print(selected_df)
Explanation:

import pandas as pd
→ Imports the pandas library for data analysis.

data = {
→ Creates a dictionary to store the dataset.

"ID": [1, 2, 3, 4, 5, 6],
→ Stores student ID values.

"Age": [18, 20, 19, 21, 22, 20],
→ Stores the age of students.

"Gender": ["Male", "Female", "Male", "Female", "Male", "Female"],
→ Stores gender information.

"Score": [45, 78, 60, 90, 35, 55]
→ Stores marks obtained.
}

df = pd.DataFrame(data)
→ Converts dictionary into DataFrame.

sorted_df = df.sort_values(by="Score")
→ Sorts data by score.

df["Final_Score"] = df["Score"] + 5
→ Computes new variable.

df["Result"] = df["Score"].apply(lambda x: "Pass" if x >= 50 else "Fail")
→ Recodes Pass/Fail.

selected_df = df[df["Score"] >= 60]
→ Selects students scoring 60 or above.
