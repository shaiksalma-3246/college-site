Experiment 2: Parametric and Non-Parametric Tests

Aim:

To perform Parametric (Independent t-test) and Non-Parametric (Chi-Square) statistical tests for analyzing data in predictive analytics.
Description:

In this experiment, parametric and non-parametric statistical tests are implemented using Python. An Independent t-test is used to compare the means of two independent groups. The Chi-Square Goodness of Fit test is used to compare observed and expected frequencies, while the Chi-Square Test of Independence is used to determine whether two categorical variables are associated.
Code:

import numpy as np
import pandas as pd
from scipy.stats import ttest_ind, chisquare, chi2_contingency


print("----- PARAMETRIC TEST : INDEPENDENT t-TEST -----")

group1 = np.array([78, 85, 88, 90, 92])
group2 = np.array([70, 72, 75, 78, 80])

t_stat, p_val = ttest_ind(group1, group2)

print("Group 1 Scores:", group1)
print("Group 2 Scores:", group2)
print("t-statistic:", t_stat)
print("p-value:", p_val)

if p_val < 0.05:
    print("Conclusion: Significant difference between the two groups.\n")
else:
    print("Conclusion: No significant difference between the two groups.\n")


print("----- NON-PARAMETRIC TEST : CHI-SQUARE GOODNESS OF FIT -----")

observed = np.array([50, 30, 20])
expected = np.array([40, 40, 20])

chi_stat, p_value = chisquare(f_obs=observed, f_exp=expected)

print("Observed Frequencies:", observed)
print("Expected Frequencies:", expected)
print("Chi-Square Value:", chi_stat)
print("p-value:", p_value)

if p_value < 0.05:
    print("Conclusion: Observed data does not fit expected distribution.\n")
else:
    print("Conclusion: Observed data fits expected distribution.\n")

print("----- NON-PARAMETRIC TEST : CHI-SQUARE TEST OF INDEPENDENCE -----")

data = {
    "Gender": ["Male", "Male", "Female", "Female", "Male", "Female", "Male", "Female"],
    "Result": ["Pass", "Fail", "Pass", "Pass", "Fail", "Fail", "Pass", "Pass"]
}

df = pd.DataFrame(data)

table = pd.crosstab(df["Gender"], df["Result"])
print("Contingency Table:")
print(table)

chi2, p, dof, expected = chi2_contingency(table)

print("Chi-Square Value:", chi2)
print("Degrees of Freedom:", dof)
print("p-value:", p)

if p < 0.05:
    print("Conclusion: Significant association exists between variables.")
else:
    print("Conclusion: No significant association exists between variables.")

1.What is a parametric test?
Answer:A parametric test assumes normal distribution and uses parameters like mean and standard deviation.

2.Give one example of a parametric test.
Answer:Independent t-test.

3.What is an independent t-test used for?
Answer:It is used to compare the means of two independent groups.

4.What is a non-parametric test?
Answer:A non-parametric test does not assume any specific data distribution.

5.Is the Chi-Square test parametric or non-parametric?
Answer:Chi-Square test is a non-parametric test.

6.What is the Chi-Square Goodness of Fit test?
Answer:It checks whether observed frequencies match expected frequencies.

7.What is the Chi-Square Test of Independence?
Answer:It determines whether two categorical variables are associated.

8.What is a p-value?
Answer:The p-value indicates the probability of obtaining the result under the null hypothesis.

9.What significance level is commonly used?
Answer:The commonly used significance level is 0.05.

10.What does p-value < 0.05 indicate?
Answer:It indicates a statistically significant result and the null hypothesis is rejected.
